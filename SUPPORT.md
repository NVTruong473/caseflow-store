# Support Policy

CaseFlow Books is a public showroom and source-evaluation project, not a
managed commerce service.

For a reproducible defect in the public showroom, open a GitHub issue without
including credentials, personal data, private order details, or security
findings. Report security findings privately as described in `SECURITY.md`.

Repository access does not include customization, deployment, migration,
provider setup, data recovery, operational support, or a service-level
commitment. Those services require a separate written buyer agreement with
explicit scope, acceptance criteria, support period, and ownership.

Feature development for the showroom is closed. Buyer-specific work starts
with the discovery and commercial handoff documents under `caseflow-store/docs/`.
