# v1.11.3 - Expert UI And Accessibility Polish

Date: 2026-07-22

Production URL: `https://caseflow-store.vercel.app`

## Summary

`v1.11.3` is a focused post-release polish patch. It improves accessibility
signals and touch-target quality discovered during a final expert audit,
without adding new commerce, auth, payment, shipping, or admin features.

## Changes

- Added explicit `id`/`htmlFor` associations to all catalog filter controls.
- Improved top support-bar link hit areas to a 32px minimum height.
- Preserved existing focus-visible states and header layout density.
- Added audit documentation and local visual/accessibility evidence for the
  patch.
