# v1.11.2 - Neutral Light UI And Pagination Patch

Date: 2026-07-22

Production URL: `https://caseflow-store.vercel.app`

## Summary

`v1.11.2` is a focused UI polish patch. It makes CaseFlow Books feel brighter
and less yellow/orange, avoiding the "Night Shift" cast that made the current
palette feel uncomfortable on some screens. It also fixes the catalog
pagination layout so large page counts no longer render as a long row of
numbered buttons that can wrap awkwardly.

This is not a redesign and does not add commerce features.

## Changes

- Shifted the body background from warm beige paper to neutral light paper.
- Changed elevated cards/forms from cream to clean white.
- Changed muted surfaces from tan to a very light green-gray.
- Softened borders so panels feel cleaner and less heavy.
- Cooled the reading-table, ledger, and cover-backdrop gradients.
- Reduced cover and soft shadow opacity slightly.
- Replaced full catalog page-number rendering with compact pagination that
  keeps the first page, last page, nearby pages, and an ellipsis.
- Added a compact mobile pagination control using the current-page summary and
  previous/next actions instead of forcing a horizontal strip of page numbers.
- Updated `DESIGN.md` and `docs/style-guide.md` so future UI work does not
  drift back toward yellow/orange Night Shift tones.
