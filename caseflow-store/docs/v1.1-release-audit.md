## Dependency Audit

Command:

```bash
npm audit --audit-level=moderate
```

Result:

- `0` critical vulnerabilities.
- `0` high vulnerabilities.
- `2` moderate vulnerabilities.
- Both moderate findings are the same transitive advisory path:
  `next@16.2.10 -> postcss@8.4.31`.
- Advisory: `GHSA-qx2v-qp2m-jg93`.
- `npm audit fix --force` proposes `next@9.3.3`, which is a breaking downgrade
  and is not an acceptable release fix.
- `npm view next version` reports `16.2.10`, so there is no newer stable Next.js
  release available from npm during this check.

Decision:

- Do not run `npm audit fix --force`.
- Treat this as a documented moderate transitive dependency risk for `v1.1`.
- Recheck after a stable Next.js release updates its PostCSS dependency.
