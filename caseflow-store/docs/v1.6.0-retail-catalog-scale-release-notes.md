# CaseFlow Books v1.6.0 Retail Catalog Scale Release Notes

- Date: 2026-07-19
- Status: production deployed and smoke tested
- Production URL: `https://caseflow-store.vercel.app`
- Vercel deployment: `dpl_AxywdtLdcWEgeC9ytoiJwqNTwCK7`
- Governing ADR:
  [`docs/adr/0011-retail-catalog-scale-and-hero-polish.md`](adr/0011-retail-catalog-scale-and-hero-polish.md)

## What Changed

- Expanded the active bookstore catalog from 100 to 500 sellable editions.
- Preserved language parity at 250 English editions and 250 Vietnamese
  editions.
- Added 400 generated retail-edition products across the existing 50 bilingual
  works: reader paperback, library hardcover, collector or annotated special
  edition, and reading-group box set.
- Generated 400 local SVG cover assets under
  `public/images/books/v16-covers`.
- Refreshed active VND price bands so no active book edition keeps an
  unrealistic low display price.
- Removed customer-facing hero copy that sounded like implementation notes and
  replaced it with bookstore signals: catalog scale, bilingual selection, COD,
  bank transfer, and account order tracking.
- Fixed catalog result-count presentation so the count remains horizontal at
  tablet/desktop widths.
- Updated release/final QA verifiers from the old 100-edition baseline to the
  500-edition baseline.

## Production Data Result

- Active editions: `500`
- English editions: `250`
- Vietnamese editions: `250`
- v1.6 generated retail editions: `400`
- New local generated covers: `400`
- Minimum active price after repricing: `120,000 VND`

## Boundaries

- The 500 products are edition variants across the existing 50 works, not 250
  unique works.
- The new generated editions intentionally omit ISBN values rather than
  inventing fake identifiers.
- Generated covers are project-created assets, not copied commercial book
  covers.
- No payment, QR settlement, shipping carrier, email, SMS, or external AI
  provider behavior changed.
