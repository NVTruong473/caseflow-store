# CaseFlow Books v1.2 Catalog Realism Baseline

Status: accepted baseline for `V12-T03` on 2026-07-17.

## Scope

This task audited the released v1.1 catalog without changing production data,
runtime behavior, schema, or public contracts. The audit compared the committed
seed with the Supabase read path, scanned the storefront source, ran a local
production build, and reviewed homepage, catalog, product detail, and
authenticated admin catalog pages at 375px and 1440px.

Evidence lives under `caseflow-store/.agent/artifacts/v12-t03/`.

## Release Blockers

1. `V12-B01` (critical): all 100 active editions use the same generic primary
   cover in both the seed and Supabase.
2. `V12-B02` (high): all 100 editions use a demo publisher and lack ISBN-13,
   ISBN-10, and page count.
3. `V12-B03` (high): 98 of 100 Vietnamese summaries contain no Vietnamese
   diacritics; populated fields therefore overstate localization quality.
4. `V12-B04` (high): the current contracts cannot attach reviewed provenance to
   works, editions, or publishers and cannot represent v1.2 merchandising
   collections.
5. `V12-B05` (medium): normal storefront source still contains 6 `TBC`, 2
   not-specified, and 15 placeholder paths.
6. `V12-B06` (medium): all 50 works have structurally valid English/Vietnamese
   edition pairs, but the catalog has no translator records or credits.
7. `V12-B07` (high): authenticated admin catalog overflows horizontally by
   195px at 375px.

Additional completeness evidence: 16 editions lack publication year; all 100
lack subtitle, table of contents, translator credits, and thickness. Fifty
English summaries and fifty Vietnamese summaries repeat edition-positioning
boilerplate.

No reviewed page had a broken image, missing image alt value, or duplicate DOM
ID in the deterministic browser heuristic. The catalog/admin unnamed-control
heuristic returned 3/2 review candidates respectively; these are not confirmed
failures until label association is checked with axe-core and assistive
technology.

The production build passed with 41 generated routes in 46.04 seconds. The
single placeholder SVG is 1,390 decoded bytes and was 300 transferred bytes in
the uncached product-detail observation. Observed CLS was 0 with no exposed
layout-shift entries. LCP was not captured by this lightweight run and remains
required in the v1.2 release quality gate.

## Schema Decision

Decision: **an additive schema change is required**.

The existing work/edition separation and cover references should be preserved.
`V12-T04` must define provenance and review contracts for source URL, checked
date, content kind, rights basis, attribution, review state, reviewer note, and
edition-match confidence. `V12-T09` should introduce only the minimum
merchandising storage needed for localized shelf labels, deterministic order,
date windows, and editorial versus sales-derived provenance.

No destructive migration, table replacement, or v1.1 data deletion is
justified by this baseline.
