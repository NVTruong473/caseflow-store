# CaseFlow Books v1.2 Canonical Catalog Manifest

Status: accepted for `V12-T05` on 2026-07-17.

## Outcome

The canonical v1.2 manifest freezes 50 real-book works and 100 sellable edition
records as 50 deliberate English/Vietnamese pairs. It is a curated import input,
not the runtime catalog yet. Database migration and replacement remain scoped to
`V12-T10` and `V12-T11`.

The manifest, build script, and deterministic report are:

- `src/data/books/v1.2-canonical-manifest.json`
- `scripts/build-v12-canonical-manifest.ts`
- `scripts/verify-v12-canonical-manifest.ts`
- `.agent/artifacts/v12-t05/canonical-manifest-check.json`
- `.agent/artifacts/v12-t05/canonical-manifest-check.md`

The category membership is intentionally multi-label. Classic literature is the
anchor category, while fiction, young readers, speculative fiction, mystery,
romance, self-development, and business/economics provide useful discovery
paths. Category counts therefore exceed 100.

## Source And Fact Rules

Every edition has one reviewed source-edition key and one stable source URL.
Facts retained from a source are represented by field-level provenance records
that all point to that same source-edition key. The build rejects mixed-edition
facts and the verifier reports unsupported facts.

Accepted source coverage is:

| Edition fact | Editions with a reviewed value |
| --- | ---: |
| Source review | 100 |
| ISBN-13 | 74 |
| ISBN-10 | 65 |
| Publisher | 65 |
| Translator | 12 |
| Publication year | 94 |
| Page count | 83 |
| Source physical format | 5 |
| Dimensions | 0 |
| Weight | 0 |

Missing optional facts remain `null`. A zero page count from a provider is also
treated as unavailable. Dimensions, weight, translator names, ISBNs, and binding
are never borrowed from another edition.

Google Books and Open Library provide most edition records. Fahasa, Minh Khai
Books, NetaBooks, Nhã Nam, Sách Tiếng Việt, and the VOV Library are limited to
specific Vietnamese editions where the broader providers did not expose a
trustworthy counterpart. Retail prices, stock labels, promotional copy, and
market availability were not imported from any of those pages.

`source-reviewed` means the source supports the selected bibliographic record.
It does not claim that CaseFlow owns cover rights, that a supplier can fulfill
the edition, or that external market availability remains current.

## Store Data Boundary

Price, compare-at price, stock, low-stock threshold, promotion eligibility,
featured state, and availability are explicitly owned by `caseflow-books` with
the basis `editorial-merchandising-decision`. They are deterministic catalog
planning data, not scraped market claims.

`store.format` is the planned CaseFlow SKU configuration. It must not be exposed
as a publisher-confirmed binding unless `bibliographic.sourcePhysicalFormat` is
also present and reviewed. This distinction prevents an inferred paperback or
hardcover label from becoming an unsupported bibliographic fact. Later
procurement work should increase exact binding coverage before production sale.

The low-stock threshold is 10 units. The manifest verifier checks that stock,
threshold, and inventory status agree, correcting an inherited seed mismatch
where some low-stock SKUs used a threshold of 5.

## Content Boundary

All English and Vietnamese summaries, localized display titles, and
merchandising rationales are project-written. No retailer description, review,
or excerpt was copied. The verifier checks bilingual completeness, Vietnamese
diacritics, and prohibited marketing/demo language.

Cover art is intentionally absent from this task. Exact edition-cover matching,
rights basis, attribution, and local derivative generation belong to
`V12-T06` and `V12-T07`.

## Compatibility Decision

The existing 49 retained works preserve all 98 edition IDs and slugs used by
public links or historical orders. `The Elements of Style` is not carried into
the sellable manifest because no trustworthy Vietnamese counterpart was found.

It is replaced in the assortment by a new pair:

- `The Old Man and the Sea`
- `Ông già và biển cả`

The replacement uses new work and edition IDs. The retired work and its two old
edition slugs use `retired-to-catalog` entries with no target redirect. Reusing
or redirecting those IDs to the Hemingway title would corrupt order history and
misrepresent old links.

## Reproduction

From `caseflow-store`:

```bash
npx tsx scripts/research-v12-edition-candidates.ts
npx tsx scripts/build-v12-canonical-manifest.ts
npx tsx scripts/verify-v12-canonical-manifest.ts
```

The research step is network-dependent and records candidates. The accepted
manifest is deterministic once the reviewed candidate artifact and explicit
source overrides are frozen. Changes to a source selection require rebuilding
and passing the verifier before the manifest can be accepted again.
