## Scope

Release, Vercel production alias, release documentation, and production smoke
evidence agree after the `v1.10.0` account-bound signup voucher release.

change schema, mutate catalog/order data, or create a new release tag.

## Not Re-Run

`scripts/verify-signup-vouchers.ts` was not re-run during this consistency
audit because the script writes to the fixed `.agent/artifacts/signup-vouchers`
directory and would overwrite the release evidence. The production signup
voucher verifier had already passed during `SIGNUPVOUCHER-T02`, and the release
documents, tag, remote branch, GitHub Release, and deployment alias were
re-verified here.
